== Installation ==

1. Upload the entire `casia-reviews` folder to the `/wp-content/plugins/` directory.
2. Activate the plugin through the 'Plugins' menu in WordPress.
3. Now there is new Gutenberg block named Casia review which You can insert in Gutenberg editor.

= Usage =
Go to [the plugin's documentation](http://vividweb.eu/get-started-with-the-casia-reviews-plugin/) for information on how to use it.
